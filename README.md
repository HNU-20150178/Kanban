# Kanban Board

## 기술 스택
- **Backend**: Spring Boot 3.2.0, Java 17, Spring Data JPA, Maven
- **Frontend**: Vue.js 3, Vuedraggable, Axios
- **Database**: Oracle Database

## 프로젝트 목적
칸반 보드를 통한 업무 관리 및 개발 효율성 향상

## 구현 완료
- 기본 CRUD (생성, 조회, 수정, 삭제)
- 드래그앤드롭을 통한 업무 이동
- 상태 관리 (TODO, IN_PROGRESS, DONE)
- 우선순위 설정 (LOW, MEDIUM, HIGH)
- CORS 설정

- Toast, loading 설정
